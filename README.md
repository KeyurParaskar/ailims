# AI-native LIMS

This project aims to build a highly configurable Laboratory Information Management System (LIMS) powered by AI, enabling lab technicians to set up, manage, and optimize workflows using natural language and drag-and-drop interfaces.

## Purpose
Streamline lab operations, reduce manual effort, and provide intelligent assistance for data management, workflow tracking, and reporting.
